DEPENDENCIES (Core)
====================

Main code of ABACUS. Modules, templates, examples and general tools are
organised in different folders. 'Dependencies (Core)' contains the 
following folders:

- Config_Examples: examples of populated configuration files, for all
  modules and available templates.
- Config_Templates: empty configuration file templates for all modules.
- Detect_Audio: code from 'Audio Detect' module
- Import_Audio: code from 'Audio Import' module
- Import_Navigation: common functions from 'Receiver Import' and 'Source
  Import' modules.
- Import_Receivers: code from 'Receiver Import' module
- Import_Sources: code from 'Source Import' module
- Initialise_Struct: functions for initialising the configuration, data 
  and database structures used throughout the software.
- Process_Audio: code from 'Audio Process' module
- Process_Navigation: code from 'Navigation Process' module.
- Various_Tools: general functions used by various modules.
